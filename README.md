# ✍️ Scriptify AI

> **Convert digital documents into realistic handwritten notes using your own handwriting or built-in handwriting fonts.**

Scriptify AI is a full-stack MERN application that transforms typed documents into natural-looking handwritten notes. Users can upload PDFs, text documents, or images, choose from multiple handwriting styles or upload their own handwriting samples, and generate printable handwritten notes on realistic notebook paper.

The project aims to bridge the gap between digital documents and handwritten notes while providing a customizable writing experience.

## ⚡ How It Works

1. Upload a document (PDF, Text, or Image).
2. Select a built-in handwriting style or upload your own handwriting sample.
3. Customize the appearance using font, paper, ink, spacing, and layout options.
4. Preview the handwritten document in real time.
5. Export the final handwritten document.

---

## 🎯 Vision

The long-term vision of Scriptify AI is to create a personalized handwriting generation platform where users can upload a single handwriting sample (A–Z, a–z, and 0–9), allowing the application to recreate documents using their unique handwriting style. This goes beyond applying fonts by reconstructing text using personalized handwritten characters.

Future enhancements include:

* AI-powered handwriting extraction
* Character segmentation
* Intelligent spacing and alignment
* Multi-page handwritten PDF generation
* Cloud storage for handwriting profiles
* Multiple saved handwriting styles per user
